## Portfolio Analysis

Perform comprehensive analysis of an investment portfolio including performance metrics, risk assessment, and optimization recommendations.

### Analysis Scope

**Performance Metrics:**
- Total return (absolute and annualized)
- Risk-adjusted returns (Sharpe ratio, Sortino ratio)
- Benchmark comparisons
- Rolling performance windows

**Risk Assessment:**
- Volatility measures (standard deviation, beta)
- Value at Risk (VaR) calculations
- Maximum drawdown analysis
- Stress testing scenarios

**Portfolio Optimization:**
- Asset allocation recommendations
- Rebalancing suggestions
- Diversification analysis
- Tax efficiency considerations

### Input Requirements

- Portfolio holdings with quantities and cost basis
- Historical price data for all assets
- Benchmark index for comparison
- Risk tolerance parameters
- Investment time horizon

### Output Format

**Executive Summary:**
- Overall portfolio health score
- Key performance highlights
- Risk exposure summary
- Action items priority list

**Detailed Analysis:**
- Performance attribution by asset class
- Risk decomposition analysis
- Optimization recommendations with expected outcomes
- Scenario analysis results

### Implementation Approach

Use quantitative finance libraries (numpy, pandas, scipy) with modern portfolio theory principles. Ensure all calculations are transparent and reproducible.
